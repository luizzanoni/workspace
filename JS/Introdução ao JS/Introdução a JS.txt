============Tipos de operadores

existem muitos simbolos, mas os mais importantes, buscar tradução na net, estes são:
atribuição, aritimietica, comparação, logica, e de condicional
atribuição:
mais comum é " = ", x=y, pronto acabou :)
+=  :  x+=y é o mesmo que x=x+y
módulo retorna o resto da divisão, simbolo do módulo é %

aritimietica, 
** é exponencial
++ incrementar, adc uma variável, i++ é o equivalente a +1
-- decrementar, adc uma variável, i-- é o equivalente a -1

comparação
== igual a
=== mesmo valor e mesmo tipo
< menor
> maior
<= menor ou igual
=> maior ou igual
!= diferente
!== valor e tipos diferentes

lógica
&& "e" logico - precisa de tudo verdadeiro para acessar 
|| "ou" logico - somente uma das infos for verdadeiro
! não logico - 

condicional (ternário)
if (condition)
	statement;
else
	statemente;
